# UI 重新设计文档 — Neon Aurora 风格

## 设计理念

本次重新设计采用 **"Neon Aurora"** 风格，借鉴了多个现代 UI 设计灵感：

- **Elegant Dark Pattern**：深邃渐变背景 + 光线条纹
- **Spotlight**：聚光灯发光效果
- **Neumorphism**：新拟态柔和阴影
- **Spline 3D**：未来科技感

## 核心设计语言

### 色彩体系

#### 背景层次（纯黑系）
- `#000000` - 纯黑（最深底色）
- `#050508` - 深空黑（面板/侧边栏）
- `#0A0A0F` - 星云黑（卡片）
- `#0F0F15` - 悬浮层
- `#14141C` - 悬停态
- `#08080C` - 输入框

#### 主强调色：极光青
- `#00FFC8` - 极光青（主色）
- `#33FFD4` - 悬停态
- `rgba(0,255,200,0.08)` - 淡背景
- `rgba(0,255,200,0.20)` - 发光效果

#### 辅助强调色：电光蓝
- `#0080FF` - 电光蓝
- `rgba(0,128,255,0.08)` - 淡背景

#### 状态色（霓虹风格）
- `#00FF88` - 霓虹绿（成功）
- `#FFB800` - 琥珀金（警告）
- `#FF3355` - 霓虹红（危险）

### 几何参数
- 按钮圆角：12px
- 卡片圆角：20px
- 输入框圆角：12px
- 图标徽章圆角：14px

### 阴影效果
- 聚光灯阴影：`0 0 30px rgba(0,255,200,0.15)`
- 柔和阴影：`0 4px 20px rgba(0,0,0,0.6)`
- 发光阴影：`0 0 20px rgba(0,255,200,0.20)`

## 设计亮点

### 1. 聚光灯效果
- 卡片边框带青色微光
- 状态指示器带发光效果
- 按钮带发光阴影

### 2. 极光青主色调
- 所有强调元素使用极光青
- 图标徽章带发光背景
- 聚焦时边框变为明亮青色

### 3. 新拟态柔和阴影
- 卡片使用多层阴影
- 按钮带发光效果
- 输入框聚焦时发光

### 4. Logo 设计
- 青色渐变背景
- 发光阴影效果
- 现代简约风格

## 文件变更清单

| 文件 | 主要改动 |

- `glow_divider()` - 发光分隔线
